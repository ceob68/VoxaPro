# Voxa Pro

Voxa Pro es una aplicación de escritorio premium para Windows que democratiza la transcripción de audio/video de manera local y privada.

## Características Principales
- **100% Local y Privada:** Procesamiento en CPU/GPU local, sin depender de la nube.
- **Compresión Inteligente:** Transcripción de videos masivos en tiempo récord mediante pre-procesamiento con FFmpeg.
- **Interfaz Élite:** Diseño moderno tipo Glassmorphism.
- **Drag & Drop:** Interfaz intuitiva y fácil de usar.

## Stack Técnico
- **Lenguaje:** Python 3.11
- **Motor ASR:** faster-whisper (CTranslate2)
- **UI:** PySide6 (Qt for Python)
- **Procesamiento de Audio:** FFmpeg
