# Voxa Pro - Guía Rápida

¡Bienvenido a Voxa Pro! Tu herramienta definitiva para transcripción local y privada.

## 1. Instalación
Simplemente ejecuta `VoxaPro_Setup_v1.0.exe` y sigue el asistente.

## 2. Primera Transcripción
1. Abre Voxa Pro desde el escritorio.
2. Arrastra tu archivo de audio o video (MP4, MP3, WAV) a la zona central.
3. Espera a que termine la extracción (la barra de progreso se llenará).
4. El texto aparecerá en pantalla.

## 3. Exportar
Una vez que el texto está en pantalla, usa los botones inferiores:
- **Guardar TXT**: Para obtener el texto plano.
- **Guardar SRT**: Para obtener subtítulos sincronizados listos para videos.

## 4. Configuración
Haz clic en el botón "Configuración" arriba a la derecha si deseas usar un modelo más rápido o cambiar a GPU.
